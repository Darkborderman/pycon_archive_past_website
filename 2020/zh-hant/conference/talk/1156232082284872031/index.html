Deploy Your Model to Live Trading

  • 藏風藝文空間
  • 第 2 天，13:50‑14:20
  • 主題分類： 金融科技
  • 語言： 中文演講/英文投影片
  • Python 難易度： 進階

摘要

近年機器學習及深度學習讓Python推上一個高峰，有許多人開始將機器學習應用於金融交易模型，但是卻缺乏將模型部署於實際交易環境快速方便地解決方案，模型訓練好後如何串接即時報價，如何高效的把當下即時資料合併歷史資料預處理成模型的輸入資料，並且如何將模型訊號與實際交易下單介接，每一步都相當複雜且有效率考量。本次演講中，會建立一套快速方便且高效率的即時交易系統，首先介紹如何使用dolphindb來將即時的串流報價資料收集，並且整合pandas與numpy等放入訓練好的模型做即時的預測，並串接到交易API進行下單，並簡單介紹如何透過dolphindb快速做模型多商品績效的回測。

說明

### 使用的第三方工具： - [DolphinDB] 為一套高效率的時間序列處理料庫，並與 pandas 高度整合，提供社群版本。 - [Shioaji] 為 Python 國內串接交易市場唯一跨平台的解決方案，主打 Python Friendly 與高效率，講者為開發者之一。 - [Pandas & Numpy] Python 資料科學的必備。 [DolphinDB](https://github.com/dolphindb/Tutorials_EN/blob/master/python_api.md) [Shioaji](https://github.com/Sinotrade/Shioaji)
