CBFHSS 專為高中生設計的管家型聊天機器人

  • R2
  • 第 1 天，13:00‑13:30
  • 主題分類： 應用
  • 語言： 中文演講/英文投影片
  • Python 難易度： 入門

摘要

CBFHSS 是以高中生生活出發所研發之聊天機器人，目的在於避免學生在每日忙碌的學校生活中忘記師長所叮嚀的作業、考試...等。本作品目前的功能包含手動新增、刪除考試、作業、段考、班級公告等重要訊息、到期作業自動刪除、放學前資訊推播及體溫登記與儲存．本作品「專為高中設計的管家型聊天機器人」 已於 2019 年 12 月 30 日於作者所屬班級啟用，透過使用者的應用與實作回饋，整個系統已漸趨穩定。因為 COVID-19 的關係，這個專案目前正漸漸的部署到全校，除了希望可以減少紙筆登記體溫時的感染風險之外，也希望學校能轉型為相對智慧的校園。

說明

### 使用過的 API (Python) - [Django 3](https://www.djangoproject.com/) - [LINE messaging API SDKS](https://github.com/line/line-bot-sdk-python) - MySQL, [MySQL Connector](https://dev.mysql.com/doc/connector-python/en/connector-python-example-connecting.html) - [APScheduler](https://apscheduler.readthedocs.io/en/stable/userguide.html) - [Google Sheet API](https://developers.google.com/sheets/api/quickstart/python) ### 使用過的雲端平臺 - [Heroku](https://www.heroku.com/) - [Google Cloud Platform](https://cloud.google.com/) - [TWCC](https://www.twcc.ai/) ### 贊助 - [NCHC 國家高速網路與計算中心](https://www.nchc.org.tw/) - 科技抗疫專案
