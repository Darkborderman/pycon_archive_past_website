重力波調查員：鄉民都來分析重力波觀測資料

  • 三道門香見
  • 第 2 天，13:50‑14:20
  • 主題分類： 資料分析
  • 語言： 中文演講/中文投影片
  • Python 難易度： 中階

摘要

黑蛋點開GW150914案發現場調查報告的第0頁：
「你們的使命不在這個星球，而我，探源小幫手，會在這30分鐘內引領你們進入案發當時的那1秒，藉由【重力波開放科學中心】提供的蒐證資料及分析工具，協助你們重現兩個緻密星體相撞的案發經過，以便完成此調查報告。鄉民們，不要只是瀏覽案件新聞看熱鬧，加入重力波調查員，夢想起飛！」

說明

由黑洞、中子星或白矮星等[緻密星體](https://zh.wikipedia.org/wiki/致密星)所組成的雙星系統，在雙星互繞合併的過程中會釋放出重力波。[GW150914](https://zh.wikipedia.org/zh-tw/GW150914)是人類首次直接探測到的重力波源，訊號持續不到1秒。天文學家藉由分析重力波觀測資料，能推算出重力波源的星體性質。「[重力波開放科學中心](https://www.gw-openscience.org/)」網站提供重力波觀測資料、軟體及教材，讓大眾能學習重力波資料分析。本演講會簡介該網站上的開放資源，並以兩個Python套件來解說重力波資料分析的步驟。 * [GWpy](https://gwpy.github.io/docs/stable/index.html)方便讓人讀取重力波觀測資料，並俱備基本的重力波訊號處理(過濾高低頻率雜訊、[傅立葉頻譜分析](https://zh.wikipedia.org/wiki/谱密度)、[時頻分析](https://zh.wikipedia.org/wiki/時頻分析))及資料視覺化功能。 * [PyCBC](https://pycbc.org/)俱備進階的重力波觀測資料分析與統計功能，可藉由[貝氏推論](https://zh.wikipedia.org/wiki/贝叶斯推断)統計方法，得到重力波源所蘊含的天文物理量值，例如：緻密星體的質量、緻密雙星與地球的距離。
