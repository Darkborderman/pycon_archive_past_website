Django, Celery 與 Windows 的美麗邂逅

  • R2
  • 第 2 天，11:55‑12:10
  • 主題分類： 應用
  • 語言： 中文演講/中文投影片
  • Python 難易度： 中階

摘要

許多 Python 套件或框架都建議使用者在類 Unix 環境(Linux, OSX)進行開發，能得到較佳的成效。然而有的時候因為專案本身的需求及限制，會需要在 Windows 環境下開發，此時可能會遇到一些沒有紀錄在官方文件上的問題。
因此，本演講將會使用簡單的範例分享在 Windows 環境開發 Django 專案並使用 Celery 遇到的一些常見問題及解決方式，以及其他因專案需求及限制無法使用官方文件建議做法的替代方案。

說明

- Django: 是ㄧ個基於MVC架構的網頁應用框架 https://www.djangoproject.com/ - Celery: 用於處理非同步任務佇列(Queue)的套件 https://docs.celeryproject.org/en/stable/index.html
