PyCon TW x PyHUG 聯合線上直播

暖身活動來到第三場，PyCon Taiwan 與地方社群 PyHUG 實體聚會進行線上與線下聯播活動。

  • 活動時間
    Wednesday, 29 July 2020 7:30 pm to 8:30 pm GMT+8
  • 活動形式
    Youtube 線上直播（ 影片連結 ）
  • 線下聚會
    Micro Beer 麥烤杯 新竹縣竹北市自強七街40號
  • 主持人
    Dainese

社群介紹: Python Hsinchu User Group

歡迎來到 PyHUG。我們是一群活動於新竹周邊的 Python 程式員。我們會定期舉辦技術討論與程式設計的聚會。非常歡迎你加入我們！
