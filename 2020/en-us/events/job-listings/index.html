Open Roles

  • 數據分析師 / 數據工程師 (風險數據)

    【職務內容】
    1. 風險數據模型建立，包含但不限於信用風險、詐欺風險、或洗錢風險等。
    2. 銀行風險資料處理與分析。
    3. 設計數據驅動風險解決方案之專案。

    【具備條件】
    1. 工作經驗：資料科學/數據分析/金融業工作經驗佳。
    2. 具SQL、Python或R程式能力。
    3. 資料庫、特徵工程、機器學習，以及大數據架構與經驗者佳。

  • 數據軟體工程師

    【職務內容】
    1. 數據介接、清洗、處理與數據模型的程式開發。
    2. 網路爬蟲、數據工程 API 程式撰寫。
    3. ETL Pipline 的工具開發與維護。
    4. 數據工程的自動化、監控、與維運模組開發。

    【具備條件】
    1. 愛好自我學習探索及分享的精神，具備團隊溝通合作的態度。
    2. 熟悉Python程式語言，並具軟體工程開發思維。
    3. 具Linux環境與Shell Script開發經驗、Git版本控管經驗、Hadoop/Hive/Spark等相關應用開發經驗。

  • 商業分析師－數據科技

    【職務內容】
    1.規劃與設計以「數據驅動」為核心之商業解決方案，並整合跨單位業務合作，扮演企業內部顧問角色，推動數據創新轉型。
    2. 分析金控或子公司之數據資料，洞察客戶行為、屬性與商機，並辨識相關風險，提出符合業務專業的顧問建議，驅動商業決策。
    3. 執行與推動分析型專案的進行，有效組織或配合專案團隊合作完成專案分析任務。

    【具備條件】
    1.工作經驗：資料科學/數據分析相關工作經驗5年以上
    2. 具SQL資料庫程式撰寫能力及 Python 或 R 資料分析程式能力
    3.熟悉資料分析、統計模型、機器學習演算法經驗者
    4.具備銀行、壽險、產險或電商等產險專業知識者佳
    5.簡報能力佳、邏輯能力佳、主動積極、抗壓性高、溝通技巧佳
