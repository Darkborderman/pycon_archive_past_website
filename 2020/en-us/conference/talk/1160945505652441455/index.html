不顧 PM 反對，工程師執意擔任 3D 美術

  • R1
  • Day 1, 16:15‑16:30
  • Category: Other
  • Language: Chinese talk w. English slides
  • Python Level: Intermediate

Abstract

人的成就感有其中一部份源自於創造，而軟體開發或程式撰寫對我而言便是在「邏輯層面」上實現創造。寫 Code 寫了一段時間之後，也許你會開始思考，能否有其他技術能創造新的東西呢？不如試著給自己一個機會，嘗試看看「視覺層面」的創作吧！

本次演講將包含開源 3D 繪圖軟體 Blender 的功能及操作簡介，並透過一些優化工作流程的小工具原始碼，淺談並示範如何在 Blender 中使用 Python 語言自行開發套件。演講最後會附上套件原始碼的 GitHub Repo 及講者聯繫資訊，歡迎找講者討論更多 Blender 技巧!
