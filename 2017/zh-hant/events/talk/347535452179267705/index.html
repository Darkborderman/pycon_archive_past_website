模仿遊戲: AI如何與人類互動獲得新技能

  • R0
  • 第 1 天，14:55‑15:40
  • Z中文演講/中文投影片
  • 科學
  • =中階

如果我們想教會機器人爬樓梯，能不能直接走一次給它看
然後機器人觀察我們的動作，自己就學會這項技能 ?

這項情境可以透過兩項技術來達成：
1. 增強學習 2. 逆增強學習

本次分享將介紹它們的概念，並呈現 python 實作成果

演講詳細內容

本次介紹的主題在機器學習領域稱為模仿學習(imitation learning) 透過逆增強學習從人類的行為，學習出目標函數，再透過增強學習，讓AI自己找出最佳的動作策略 由於深度學習的活躍，近期也出現許多互相融合的有趣成果 本次分享內容將介紹如何結合深度學習與模仿學習 並透過物理模擬引擎MuJoCo呈現機器人的學習成果，還有講者在理解與實作上的歷程心得

講者介紹

Jiawei Chen

MLDM Monday與 Kaohsiung Python社群 Co-organizer, 目前在學術單位擔任研究人員，熱衷於研究機器學習相關技術，喜歡分享相關技術，並應用在實務工作上。
