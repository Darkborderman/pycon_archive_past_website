聽風者 - 使用 GNU Radio 分析風的聲音

我們生活所在的空間充滿了訊號。

要怎麼理解這些看不到、也聽不到的訊號呢？

GNU Radio 是知名的(無線)訊號處理開源軟體，也可以使用 Python 來擴展它的處理、分析能力。

藉由個案分享，來讓社群朋友認識它。

演講詳細內容

打算以 無線鍵盤 作為個案，來探討一點(非公開的)通訊協定外，以及如何使用 GNU Radio 來觀察、分析、解譯這些訊號。
