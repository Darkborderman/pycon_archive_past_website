想像我們飛過臺北的天空，欣賞梵谷和大師們筆下的臺北！

Deep Learning 讓這件事變得可能！
這次短講，我們想和大家分享深度學習如何讓電腦學會大師的筆觸和風格(Convolutional Neural Networks, CNN)。以及我們如何結合在地特色實作的成果。

關於背景知識，並不設限，我們會從簡單的概念開始談起，如果有基本的數學認識尤佳。
