PoS: 環保又安全的區塊鏈共識機制

  • R2
  • 第 1 天，17:20‑17:50
  • Z中文演講/中文投影片
  • 金融科技
  • =中階

共識機制是區塊鏈運作原理的核心。自比特幣以來，大多區塊鏈使用工作量證明（Proof of Work, PoW）作為共識機制，運作的原理會消耗大量水電等天然資源。目前 Ethereum 生態系的開發者正在開發權益證明（Proof of Stake, PoS）的演算法，可避免 PoW 不環保的缺點並有額外安全性好處。我們將簡介官方的幾個 Python 實作，以利有興趣的開發者參與。
