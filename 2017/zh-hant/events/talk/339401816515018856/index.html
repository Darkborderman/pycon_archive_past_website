土炮股票分析系統

  • R1
  • 第 2 天，16:30‑17:15
  • Z中文演講/中文投影片
  • 金融科技
  • –入門

主要是介紹如何使用 Python 簡單的做股票的量化分析，包括資料擷取、選股、策略回測等，會以美股為例來實作。公司資料擷取的部分，會以 Pandas 來做說明，歷史股價的擷取則會使用 Pandas 跟 Quandl 做說明，選股策略的部分會實作波動率選股，進出場策略實作突破策略，回測的部分使用 Sharpe Ratio 並計算 Maximum Drawdown (MDD)。

演講詳細內容

主要是介紹如何使用 Python 簡單的做股票的量化分析。 大綱如下： 1. 簡介 股票投機賺錢是藝術、還是科學？ 介紹一下華爾街傳奇 - 海龜交易法則！ 2. 投機致富三大要件 (3Ms) 說明投資心理(Mind)、資金控管(Money)、跟系統(Method)的重要性。 3. 系統架構 說明DIY分析系統需要哪幾個部分。主要是抓資料、訂定策略(選股策略、進出場策略)、回測... 4. 使用 Python 抓股市資料 包含： 公司基本資料：使用 Pandas 做說明。 歷史股價資料：使用 Quandl 及 Pandas 來做說明。 5. 選股 使用簡化的波動率選股來做說明，並用 Pandas 實作。 6. 策略 實作簡單的突破策略。 7. 回測 介紹 Sharpe ratio，並用 Python 來實作策略回測。 8.
