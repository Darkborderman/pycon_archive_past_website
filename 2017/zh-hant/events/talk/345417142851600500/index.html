遺傳演算法最佳化高頻交易策略

  • R0
  • 第 2 天，13:25‑13:55
  • Z中文演講/中文投影片
  • 金融科技
  • =中階

應用遺傳演算法特性最佳化股市投術分析參數，利用高頻交易的tick股票資料，以不同的均線(5、10、20)，和幾種技術指標，找出每檔股票的均線與技術指標參數的最佳化。並搜尋出資金配置的最佳停損、停利點。

演講詳細內容

資料來源 資料: 台灣上市上櫃股票846 週期: Tick 時間: 20160101~20161230 內容: 股票代號、交易時間、價格、成交量 工具 1.自動載入Data : 永豐銀行 API 2.資料庫 : MySQL 3.遺傳基因演算法分析 : Python 套件deap 1.0.2進行分析 4.介面 : 用Django_Web呈現 適應函數: 報酬率/最大虧損 編碼: 實數編碼 回測KPI 交易次數 最大報酬 最小報酬 平均報酬 連續虧損次數 連續虧損金額 獲利因子 獲利金額 賺賠比 期望值 年化報酬率 勝率 ...
