衝刺開發（Sprints）

活動簡介

Sprint 活動安排在會議三天以外的時間。當天的 sprint 將有開源專案的負責人，帶著他們專案待解決的問題、issue 來現場與大家分享與解說，參加的人可以就自己感興趣的專案一同加入。透過這個互動機會將能更了解自己關注的相關專案領域。歡迎帶自己的開源專案來找同好。對已經有一定 Python 基礎的人，想參加開源專案開放但不知如何加入，或者害羞怕自己不了解該專案環境的人，參加 sprint 是個大展身手的好機會！

對於沒有把握自己是否能加入開發的朋友，四月的活動也會有 workshop 形式的工作坊，歡迎一起來玩！

審稿系統優化計畫開跑！

六月份：PyCon Taiwan x Taipei.py x Keynote 見面會

一起加入 Keynote Speaker 的專案計畫吧！

主議程活動結束後是否還意猶未盡呢？想起各種有趣的技術想一展身手？或是想與同好一起解決發現的 bug 與改進專案的程式碼？會議結束一天讓我們延續著興奮與熱情的心一起來玩專案。

PyCon Taiwan 2017 的基調演講講者 Russell Keith-Magee 將受邀光臨我們的六月份衝刺發開活動現場一起與大家分享他的專案囉！

不知道大家有聽過 BeeWare 這個專案嗎？BeeWare 是一系列可以用來幫助幫助開發、調試和啟動 Python 的工具集合。每個工具都遵循 Unix 工具的設計哲學：「一個工具只單獨將一件事情做好」。儘管自稱是整合開發環境 (Integrated development evironment, IDE)，這系列工具的概念不同於傳統的 IDE，每個工具是各自獨立的，可以單獨使用。這個 Project 的一大貢獻在於，將 Python 帶進行動裝置的平台上。相信作為一個 Python 的熱愛者，你聽到這裡一定心動了吧！快到 BeeWare 的官方網站來一探究竟然後一起加入本次的衝刺開發吧。
