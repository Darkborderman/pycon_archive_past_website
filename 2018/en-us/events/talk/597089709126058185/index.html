Python工業4.0 - 自造工廠監控系統

  • Location: R2
  • Slot: Day 1, 11:25‑12:10
  • Category: Internet of Things
  • Language: Chinese talk w. Chinese slides
  • Python Level: Intermediate

Abstract

「工業4.0」這個口號已經多俯拾即是? 資料科學家都跟你說：「把資料傳上來，我們就能幫你鍊成黃金!」?
但是當你在身處工廠中，看著加工的機械、奮力工作的工人們時，是否想到一個根本的問題?
資料到底要怎麼得到呢?
為何參加很多雲端應用的研討會都只教你怎麼「處理資料」、「分析資料」，但是就是不跟你說怎麼「得到資料」。
本演講將說明如何建立一個工廠監控系統，使用Python語言和冷冰冰的機器對話，藉由工業通訊協定、Sensor來收集工廠資訊，進而呈現一個即時監控的系統。

Description

### 使用的Python package及硬體工具 - [modbus-tk] 為Modbus套件，實作Modbus/RTU及Modbus/TCP通訊協定 - [flask] 為輕量級web framework - [matplotlib] 為Python中極好用的繪圖函式庫 - [PLC] 使用國產的永宏PLC為例 - [電表] 使用士林電機之電表為例 [modbus-tk]: https://github.com/ljean/modbus-tk [flask]: http://flask.pocoo.org/ [matplotlib]: https://matplotlib.org/ [電表]: http://www.seec.com.tw/Content/Goods/GList.aspx?SiteID=10&MmmID=655575436061073254&CatId=2015120316233269372 [PLC]: http://www.fatek.com/tw/

Speaker

由機構工程師轉職的韌體工程師，對IoT的議題感到很有興趣，平時工作因為大多接觸的是較低階的硬體，因此主要使用C語言，近二年開始接觸Python語言後，開始逐步把工作上的需求轉用Python來完成，希望在社群和更多同好一起開心的探索工業控制和IoT的領域。
