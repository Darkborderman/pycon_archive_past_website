How Pythoneers Stop Worrying About Excel Reports

  • R1
  • 第 3 天，11:30‑12:00
  • 主題分類： 資料分析
  • 語言： 中文演講/英文投影片
  • Python 難易度： 入門
  • 不錄影

摘要

Python人人都在練，Excel人人要面對，那麽能不能利用Python，讓所有人手上的Excel工作做得更快更好？這個演講新手老手皆宜，研發行銷會計製造通通來，人人有功練。

對於Excel經驗大於Python經驗的人，演講裡會提供相同於Excel操作邏輯的作法，無痛上手，又可達到自動化功效；
對於Python經驗大於Excel經驗的人，演講裡會提供耳熟能詳python library（相信你早已裝了）的作法，大幅超車「用Excel做Excel」的侷限；
對於毫無Python經驗只是路過的人，對於初學Python覺得無聊想放棄的人，也許你會認真考慮這個能讓你更早下班下課的工具；

演講中不只介紹方法，更是介紹方法論，讓每個人可依據背景與任務需求，用不同的Python實作方式完成目標。演講中不只介紹方法，更是介紹方法論，讓每個人可依據背景與任務需求，用不同的Python實作方式完成目標。演講中輔以開放資料實例說明，讓聽眾可以很容易複製講者的操作。這些都是務必可以期待的收穫。
