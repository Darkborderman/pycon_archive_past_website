Py 車達人

  • R0
  • 第 1 天，15:45‑16:15
  • 主題分類： 物聯網
  • 語言： 中文演講/中文投影片
  • Python 難易度： 中階

摘要

當我們便利的使用車輛於日常交通時，你知道車上有一台行車電腦在和它的周邊不斷通訊嗎?
當保險業者說要由你的行車資訊來決定你的保險額度、費用時，你知道除了AI在分析你的開車習慣以外，是什麼把你的車輛資訊取出來送回去的嗎?
本演講將說明行車電腦常用的CANBus協定，並使用Python語言藉此協定把車輛資訊取出上傳至雲端，讓資料科學家來分析數據。

說明

- CANbus為汽車產業中常用的標準通訊介面，目前亦被用於其它需要高可靠性傳輸的工作控制應用中 - Python3.4以上原生支援Socket-CAN，較為方便使用；而以下版本，如Python2.7時，須改用C來實作訊介面API，亦將分享這部份的實作經驗 - 雲端使用dweet.io / freeboard 來快速完成dashboard - 並進行實際應用案例之分享說明

講者

由機構工程師轉職的韌體工程師，對IoT的議題感到很有興趣，平時工作因為大多接觸的是較低階的硬體，因此主要使用C語言，近二年開始接觸Python語言後，開始逐步把工作上的需求轉用Python來完成，希望在社群和更多同好一起開心的探索工業控制和IoT的領域。
