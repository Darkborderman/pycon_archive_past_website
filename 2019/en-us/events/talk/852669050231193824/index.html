你到底匯還是不匯？（當爬蟲遇上外匯…）

  • R0
  • Day 3, 10:50‑11:20
  • Category: FinTech
  • Language: Chinese talk w. English slides
  • Python Level: Novice

Abstract

介紹外匯與其市場, 以及匯率在金融科技公司裡扮演的角色.
日常工作中, 金融交易操作者(Treasury)必須隨時監控來自多管道的資訊 透過低買高賣與遠期交易等操作將公司財務風險分擔. 今日這些資訊多數已可依靠專業軟體的輔助來獲得, 如彭博(Bloomberg), 路透(Thomson Reuters)等大型企業有能力整合來自各國權威機構或一線銀行,企業的消息, 但新創公司所面向的客戶並不全然屬於此類機構, 多數的合作對象更可能是偏向在地性質的二三線銀行, 或技術能力有相對落差的在地金融服務供應商. 此時便有機會利用 Python 程式自動化將這些對象的資訊整合, 進而強化內部企業效率.

Description

外匯(Foriegn Exchange)部分: 1. 簡介外匯/匯率 a. 一國貨幣的強弱反映其經濟的興盛衰敗 b. 兩國貨幣的交換行為, 即稱外匯 c. 兩國貨幣的交換比例, 即稱匯率 2. 外匯市場及匯率與其他金融商品(債券/股票)之不同. a. 全球最大金融(有市無場)市場 & 24H皆可買賣 b. 成交量極大, 不易受大戶影響. 3. 外匯操作的費用種類 4. 操作匯率的目的與手法 (套利或是避險) a. 實盤交易(現貨): 以足額金錢交換他國貨幣, 並持有足夠的時間來賺取匯率價差. b. 需盤交易(合約): 以合約的方式, 透過操作槓桿, 來賺取匯率價差. c. 匯率避險/套利(補拋利率套利)範例 Python部分: 1. 網路爬蟲該如何應付不同國家銀行的牌告匯率呈現方式. (東南亞各國的銀行機構, 使用的的網頁技術落差大) a. 簡單的靜態網頁 (Beautifulsoup) b. 動態產生的表格 (Selenium) c. 檔案形式 (PDF/CSV) 2. 如何利用 Python WebSocket 套件與瀏覽器支援的 WebSocket API 將爬蟲取得的匯率資料即時顯示在監控頁面 (Python <=> JavaScript).
