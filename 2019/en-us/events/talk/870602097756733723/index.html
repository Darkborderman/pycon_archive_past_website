用 Python 拉近群眾與星空的距離：Astrohackers in Taiwan 社群介紹

  • R0
  • Day 3, 13:00‑13:30
  • Category: Community
  • Language: Chinese talk w. Chinese slides
  • Python Level: Novice

Abstract

每當黑蛋仰望星空，他總幻想能伸手捕獲星星來一探其奧秘，但即便看了科普書籍及最新天文研究的新聞報導，他仍覺得離星空好遠。

某天，黑蛋在搜尋引擎輸入"開放天文資料"，發現一個以遊戲的方式來引導玩家使用開放天文資料的網站，他就此進入了精神時光屋。在他完成「以爬說語指引蛇蛇吃星星」的任務後，收到一則來自Astrohackers-TW的訊息:
「哈囉，這是Astrohackers-TW對等級0的天文黑客們的呼召，我們在尋找潛在的天文黑客，與我們一同對抗試圖把天文資源佔為私有的邪惡勢力。接受呼召前來參加天文黑客松的成員們，將一同合作創造出有助於天文教育與研究的開源軟硬體，成為推動全民共享及再創造天文知識的一份子。以下是天文黑客松的報名資訊...」黑蛋受到感召，前往參加天文黑客松，於是黑蛋破殼了，成為等級1的天文黑客，並藉由一次又一次地協作開源天文專案，在公民天文學家之路上，持續升級。

這是關於Astrohackers in Taiwan社群的現在及未來式...

Description

"""講者為「[Astrohackers in Taiwan](https://github.com/Astrohackers-TW)」社群的共同創辦人，我們致力於以黑客精神和遊戲化方式推動天文研究的開放。""" ```Python from astrohackers.tw.IANCU import pythonadventure pythonadventure.__doc__ ``` 「[Astrohackers-TW: 中央大學天文所Python使用者討論會](https://www.facebook.com/groups/1022708484514663/)」為Astrohackers in Taiwan和中央大學天文所成員共同組織的Python使用者FB社團，讓對於Python在天文研究上的應用有興趣的人彼此交流。 ```Python from astrohackers.tw import astrohackathon astrohackathon.__doc__ ``` 「[Astrohackers-TW: 天文黑客松](https://www.facebook.com/groups/888264928001934/)」為Astrohackers in Taiwan與教研/社群單位合作舉辦的天文黑客松活動，藉由這種協作活動，讓參與者學習如何將Python應用在天文教育與研究上，以及一同開發出有助於天文教育與研究的開源軟硬體工具、教材。目前與我們合作的單位有: 國立中興大學附屬高級中學。 ```Python from astrohackers.tw import nearfuture nearfuture.__doc__ ``` Astrohackers in Taiwan將在不久的未來開發出以下新模組： * 建立部落格發表關於開放天文的文章，內容包含： * 公開讓大眾取用的天文資料介紹 (例如, [黑洞影像資料](https://eventhorizontelescope.org/for-astronomers/data)、[重力波資料](https://www.gw-openscience.org/data/)) * [Astrophysics Source Code Library](http://ascl.net/)上的Python專案介紹 (例如, 太陽系外行星資料分析工具[Lightkurve](https://ascl.net/1812.013)、黑洞影像分析工具[ehtim](http://ascl.net/1904.004)、重力波資料分析工具[PyCBC](https://ascl.net/1805.030)) * 天文黑客松活動期間所開發出的專案介紹 * [遊戲化(Gamification)](https://en.wikipedia.org/wiki/Gamification)在科學教育及傳播的應用 * 建立能引導大眾使用開放天文資料的遊戲化平台，以做為公民天文學家的基礎訓練教材 我們社群如剛進入到[主序帶](https://zh.wikipedia.org/wiki/主序星)的年幼恆星，歡迎其他社群多多"imports" :)
