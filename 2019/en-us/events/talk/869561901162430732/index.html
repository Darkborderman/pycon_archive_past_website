Python 在 Data for Social Good 領域常見的應用

  • R0
  • Day 1, 14:50‑15:35
  • Category: Data Analysis
  • Language: Chinese talk w. Chinese slides
  • Python Level: Novice

Abstract

Data For Social Good(D4SG)是從2015年發起的一個用資料做公益的計畫，協助了超過30個NGO及公部門單位，並獲得總統府黑客松前五強，及聯發科的「智在家鄉」第一名等殊榮。在30多個專案中涵蓋影像辨識、文字探勘及資料分析，Python是常用的程式語言之一。本次Talk將分享給Python初學者在哪些情境可以如何做資料分析、機器學習來解決社會人文相關題目。

Description

在過去四年Data for Social Good累積超過30個專案 (http://d4sg.org/fellowship/projects/), Python為其中主要資料分析及機器學習的語言。想透過這30個專案的成果來分享Python常見運用狀況及情境。
